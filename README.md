# PlantMap

Приложение для учета растений на территории питомника. Позволяет вести базу растений, размещать их на интерактивном плане, отслеживать обработки и подкормки, выполнять поиск и получать статистику.

**Версия:** Release-3.3 (versionCode 54)  
**Дипломный проект** | 2026

---

## Возможности

- **Интерактивный план**  
  Размещайте точки с растениями на плане территории. Поддерживается масштабирование (pinch-to-zoom) и панорамирование.

- **База растений**  
  Встроенная база с возможностью добавления, редактирования и удаления растений. Каждое растение содержит название, тип, группу, цвет цветка, дополнительную информацию и список доступных объемов горшков.

- **Учёт ухода**  
  Для каждой точки фиксируются даты последней обработки и подкормки. Наглядные индикаторы позволяют быстро оценить «забытые» растения.

- **Поиск и фильтры**  
  Поиск по названию, типу, группе, цвету, объему горшка и другим параметрам. Результаты поиска подсвечиваются на плане.

- **Статистика**  
  - Общее количество растений и количество растений, соответствующих заданным критериям отбора
  - Растения без обработки/подкормки (в том числе дольше N дней) с возможностью перехода к результатам на плане

- **Экспорт и импорт базы**  
  Выгрузка БД в файл с временной меткой и возможность «поделиться» через стандартное меню Android.  
  Импорт актуальных данных о растениях с Яндекс.Диска (JSON).

- **Поддержка двух тем**  
  Автоматически подстраивается под светлую или темную тему устройства. Все цвета и стили вынесены в ресурсы, обеспечивая консистентный вид в обеих темах.

## Структура проекта
```
com.example.plantmap
├── model/ 		      # Модели данных (Plant, PlantPoint, FlowerColor и др.)
├── db/ 		      # Работа с БД
│ ├── dataaccess/ 	  # Прямые запросы (PlantDataAccess, PointDataAccess)
│ └── yandex/ 		  # Взаимодействие с Яндекс.Диском (загрузка фото, обновление БД)
├── plant/ 		      # Репозиторий, адаптер, диалоги и форма для растений
├── plan/ 		      # План территории (PlanView, EditMode)
├── search/ 		  # Поиск точек (SearchEngine, SearchDialog)
├── stats/ 		      # Статистика
├── ui/ 		      # Activity и Fragments (MainActivity, PlanFragment, DbFragment, StatsFragment)
└── util/ 		      # Вспомогательные утилиты (клавиатура, валидация, layout, даты)
```

## Сборка и запуск

### Способ 1: Установить готовый APK (рекомендуется для проверки)

1. Перейдите в папку `plantmap/app/release`.  
2. Скачайте файл `app-release.apk` на устройство Android.  
3. Откройте файл и подтвердите установку (возможно, потребуется разрешить установку из неизвестных источников).  

APK подписан и готов к использованию. Версия: **Release-3.3** (код 54).

### Способ 2: Собрать из исходников

1. Клонируйте репозиторий.  
2. Откройте проект в Android Studio (Gradle синхронизируется автоматически).  
3. Убедитесь, что установлен SDK 36 (через SDK Manager при необходимости).  
4. Запустите на устройстве или эмуляторе с Android 10 (API 29) и выше.

## Особенности реализации

- **Координаты плана** хранятся в эталонных единицах (не зависят от плотности экрана). При первом запуске выполняется миграция старых координат под новую плотность (2.8125 → 1.0).
- **База данных** поставляется в assets и копируется при первом запуске или обновлении. Версия проверяется через `PRAGMA user_version`.
- **Работа с Яндекс.Диском**:
  - Прямые ссылки на изображения получаются через `YandexDiskHelper`.
  - Обновление списка растений происходит через `UpdateDatabase`, который скачивает JSON, парсит и синхронизирует таблицу `plants`.
- **Поиск** не зависит от БД: фильтрация выполняется в памяти через `PlantSearchEngine`.
- **Темы**: приложение использует `AppCompat` и Material Design, что обеспечивает автоматическое переключение между светлой и тёмной темой в зависимости от системных настроек.
